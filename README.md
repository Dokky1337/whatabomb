# What'a Bomb!

A fast-paced, explosive multiplayer Bomberman-style game built with TypeScript and Babylon.js!

## Features

- **Multiple Game Modes**: 1v1, 1v2, 1v3 against AI, or local PvP
- **12 Arena Maps**: 3 sizes (small, medium, large) x 4 themes (classic, ice, lava, forest)
- **Smart AI**: Difficulty-based AI with escape pathfinding
- **Power-Ups**: Extra bombs, larger blasts, kick, throw, and speed boosts
- **Statistics & Achievements**: Track your progress and unlock achievements
- **Customization**: Player colors, difficulty settings, and more

## Controls

### Player 1
- **WASD** - Move
- **Space** - Place/Throw Bomb

### Player 2 (PvP Mode)
- **Arrow Keys** - Move
- **Enter** - Place/Throw Bomb

### General
- **Escape** - Pause/Resume Game

## Power-Ups

- **Extra Bomb** (Blue) - Increase bomb capacity
- **Larger Blast** (Yellow) - Bigger explosion radius
- **Kick** (Brown) - Kick bombs by walking into them
- **Throw** (Peach) - Throw bombs 3 tiles away
- **Speed** (Cyan) - Move faster ï¿½

## Development

### Prerequisites
- Node.js 18+
- npm

### Setup
```bash
npm install
```

### Development
```bash
npm run dev
```

### Production Build
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Tech Stack

- **TypeScript** - Type-safe JavaScript
- **Babylon.js** - 3D rendering engine
- **Vite** - Build tool and dev server
