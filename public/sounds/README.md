# Sound Files

Place your sound effect files here:

## Required Sounds:
- `bomb-place.mp3` - When placing a bomb
- `explosion.mp3` - Bomb explosion
- `powerup.mp3` - Collecting power-up
- `victory.mp3` - Winning the game
- `defeat.mp3` - Losing the game
- `bgm.mp3` - Background music (looping)
- `menu-music.mp3` - Menu background music

## Free Sound Resources:
- https://freesound.org
- https://opengameart.org/art-search?keys=bomb
- https://kenney.nl/assets (Game Audio packs)
- https://mixkit.co/free-sound-effects/

## Tips:
- Keep files small (under 100KB for SFX)
- Use MP3 or OGG format
- Normalize volume levels
- Background music should loop seamlessly
