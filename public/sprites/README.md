# Sprite Folder

Place your character sprite sheets here:

- `player1.png` - Player 1 character sprite sheet
- `player2.png` - Player 2 character sprite sheet  
- `enemy1.png` - Enemy type 1 sprite sheet
- `enemy2.png` - Enemy type 2 sprite sheet
- `enemy3.png` - Enemy type 3 sprite sheet

## Recommended Sprite Sheet Format

- **Size**: 32x32 or 64x64 pixels per frame
- **Layout**: Grid layout with walk cycles
- **Animations needed**:
  - Walk Down (4 frames)
  - Walk Up (4 frames)
  - Walk Left (4 frames)
  - Walk Right (4 frames)

## Where to Find Sprites

Free sprite resources:
- https://opengameart.org
- https://itch.io (search "sprite sheet")
- https://kenney.nl/assets
- https://craftpix.net/freebies/

Search for: "top-down character sprite sheet" or "bomberman style sprites"
